The browser cache is a collection of downloaded data from visited websites. The idea is that instead of downloading the same data over and over again, it can instead be loaded much faster from the browser cache. Cookies are files downloaded by the web browser that normally store user-specific information so that you don't have to re-enter it every single time you revisit a site. These entities need to be cleared for certain widgets to work (such as Mailchimp) because if they are not cleared, Mailchimp acts as though it has already been displayed to you and therefore does not need to popup again.
